# Chrome Web Store Listing — xTil

## Name (max 30 chars)
xTil — AI Page Insights

## Short Description (max 132 chars)
Extract the signal, distill the insight. AI summaries of articles, PDFs, videos & PRs with diagrams, chat and Notion export.

## Detailed Description

xTil (pronounced "ex-til", from extract + distill) turns any web page into structured, actionable knowledge — not just a summary, but a full breakdown you can refine, discuss, and share.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Stop reading 3,000-word articles. Get structured insights in seconds.

xTil turns any web page, PDF, video, or PR into a multi-section analysis — key takeaways, fact-checks, diagrams, and more. Then refine with chat and export anywhere.

────────────────────────────────────────

WHAT YOU GET
Every page produces a structured breakdown: concise overview, key takeaways with bold labels, notable quotes with original language preserved, a fact-check grading claims as verified/contested/unverifiable, pros & cons, and a conclusion. LaTeX math renders beautifully — perfect for academic papers. Choose between brief, standard, or detailed analysis — xTil automatically scales the depth to match the content.

────────────────────────────────────────

PLATFORM AWARENESS
Every platform gets special treatment:
• PDFs — Summarize academic papers, reports, and any PDF opened in Chrome, with figure extraction
• YouTube — Full transcript summary with clickable timestamp links
• GitHub — Six modes: PRs, issues, code files, repos, commits, and releases
• Reddit — Complete threads with nested comments, scores, and engagement metrics
• Twitter/X — Thread reconstruction with engagement data and notable replies
• Google Docs — Reads content directly from your authenticated session
• Facebook — Extracts posts from feed overlays with reaction counts
• LinkedIn — Summarize posts from feed or direct URLs with engagement metrics
• Gmail — Summarize emails and threaded conversations from the reading pane

────────────────────────────────────────

AUTO-GENERATED DIAGRAMS & IMAGE ANALYSIS
xTil generates Mermaid diagrams that visualize the content's structure — flowcharts, timelines, and more, with automatic light/dark theme support. Pages with images get vision-powered analysis: charts, screenshots, and infographics are read and woven into the summary.

────────────────────────────────────────

BRING YOUR OWN KEY
No subscription. Use the AI you already pay for. Works with all major LLM providers and any OpenAI-compatible endpoint, including self-hosted models. xTil auto-discovers available models and probes vision capability — image analysis just works.

────────────────────────────────────────

REFINE WITH CHAT
Ask follow-up questions, request deeper analysis, or explore specific sections. Use web search to fact-check claims against live sources. Chat responses are structured with the same quality as the initial summary.

────────────────────────────────────────

EXPORT ANYWHERE
Copy as Markdown, export to Notion with native equation support, or print to PDF. Your summaries go where you need them.

────────────────────────────────────────

MAKE IT YOURS
• Light, dark, or system theme — diagrams adapt automatically
• Summarize in any language, or match the source
• Three detail levels: brief, standard, or detailed

────────────────────────────────────────

PRIVACY FIRST
No backend, no proxy, no middleman. Content goes directly from your browser to your chosen LLM provider using your own API key. No accounts, no analytics, no tracking, no servers. Open source — verify every line at github.com/aitkn/xtil.

────────────────────────────────────────

Free. No account needed. Just install and add your API key.
